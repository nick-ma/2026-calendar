# Fonts Directory

This directory contains beautiful fonts for the 2026 calendar project.

## Chinese Fonts
- **Songti.ttc** - Elegant serif font (recommended for traditional look)
- **STHeiti Medium.ttc** - Modern sans-serif font (recommended for modern look)

## English Fonts
- **Baskerville.ttc** - Classic elegant serif
- **Futura.ttc** - Modern geometric sans-serif
- **Athelas.ttc** - Modern refined serif
- **Didot.ttc** - Luxury high-end serif

## Recommended Combinations

### Option 1: Elegant & Traditional
```bash
--font-cn fonts/Songti.ttc
--font-en fonts/Baskerville.ttc
```

### Option 2: Modern & Clean
```bash
--font-cn fonts/STHeiti Medium.ttc
--font-en fonts/Futura.ttc
```

### Option 3: Refined & Sophisticated
```bash
--font-cn fonts/Songti.ttc
--font-en fonts/Athelas.ttc
```

### Option 4: Luxury & High-End
```bash
--font-cn fonts/Songti.ttc
--font-en fonts/Didot.ttc
```
